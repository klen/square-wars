# Square Wars

A game for [Pico8](https://www.lexaloffle.com/pico-8.php) virtual console.

[Lexaloffle BBS Post](https://www.lexaloffle.com/bbs/?tid=144970)

## How to play

- Use arrow keys to choose the color.
- Press `Z` to apply.
